This is a Pomodoro timer made out of the Tkinter python module. It is one of my projects to get started in object-oriented programming. The timer can easily adjusted as the work time and break times were made as contants at the very top and as inputs to the function calls

Pong is a game created using the Tkinter Python module. I needed multiple Python files to not clog a single notebook with codes. Python files can be imported alongside the variables and functions associated. And these imported functions can be used to initiate and operate Class defined objects. While loops was used to make the ball appear in every frame update, which was possible because of Screen tracer method. You can set the tracer to 0 to completely blank out the screen, and then used time module to control when and how fast you the screen to reappear to hte latest of movement of each object.

Snake game was my introduction to visual OOP. It functions almost identically to the Pong game. You can create separate files for every object making up the game: blocks of squares making up the snake, food that randomly appears each encounter with the snake, Scoreboard that updates each successful encounter, while all operating inside the MAIN.PY file. The main important module that makes all of this possible is the turtle module. From the module specific constructors where used, the Screen constructor for the screen and the Turtle constructor for the blocks of squares that makes up the snake, as well as the scoreboard that prints the scores.
